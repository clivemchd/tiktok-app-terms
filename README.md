## Purpose

This site exists to satisfy the [TikTok Developer Platform](https://developers.tiktok.com/) app review requirements:

- **Privacy Policy** with third-party tracking disclosure, opt-out mechanisms, children's privacy, and direct links to TikTok's Privacy Policy, Cookie Policy, and Business Products Data Terms
- **Terms of Service** with acceptable use referencing TikTok Community Guidelines, Developer Terms, IP handling, indemnification, and limitation of liability
- **Fully developed website** demonstrating the application's purpose and legitimacy

## Key Points

- **Personal-use only** — single operator, no public users, no sign-ups
- **No data collection** from third parties — only the operator's own API tokens are stored
- **No tracking** — no analytics, no pixels, no cookies beyond platform defaults
