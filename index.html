Shorts Pipeline

An AI-powered content creation tool that converts comic book pages into short-form video with “Did You Know” narration — published across TikTok, YouTube, and Instagram.

About This Application

Shorts Pipeline is a content creation tool built by Moneounchpan (@moneounchpan on TikTok) that automates the process of creating short-form video content from comic book pages and publishing it across social media platforms including TikTok, YouTube, and Instagram.

How It Works

Extract

Comic book pages are extracted from PDF, CBZ, or CBR files.

Analyze & Script

AI vision models analyze the pages and generate "Did You Know" narration scripts.

Produce

Text-to-speech narration is generated and composed with the comic panels into short-form video.

Publish

Videos are published to TikTok, YouTube, and Instagram via their official APIs.

Key Facts

Automated Pipeline

The entire workflow from page extraction to multi-platform publishing is fully automated — no manual steps required.

🔒

No Data Collection

No personal data is collected from third parties. The only data processed are API authentication tokens for content publishing.

Original Content

All videos are original creations featuring AI-generated narration about comic book facts and trivia.

✅

Platform Compliant

Content is created in compliance with TikTok Community Guidelines, YouTube Terms of Service, and Instagram Terms of Use.

Third-Party Services Used

This application integrates with the following third-party services for content creation and publishing:

  • TikTok Content Posting API — Publishing videos to TikTok (@moneounchpan)
  • YouTube Data API v3 — Uploading and scheduling videos on YouTube
  • Instagram Graph API — Cross-posting videos as Reels on Instagram
  • Google Cloud Text-to-Speech — Generating voice narration for videos
  • Google Gemini API — AI vision analysis and script generation
  • Cloudinary — Temporary video hosting for Instagram upload processing
