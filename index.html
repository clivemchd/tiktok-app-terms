Shorts Pipeline

A personal-use automation tool that converts comic book pages into short-form video content with AI-generated "Did You Know" narration — created and operated solely by Moneounchpan.

Personal Use Only — Single Operator — No Public Users

About This Application

Shorts Pipeline is a personal-use application built and operated exclusively by Moneounchpan (@moneounchpan on TikTok). It is not a public service, does not have any registered users, and does not accept sign-ups. The sole operator is also the sole user.

The application automates the process of creating short-form video content from comic book pages and posting it to the operator's own social media accounts, including TikTok, YouTube, and Instagram.

How It Works

Extract

Comic book pages are extracted from PDF, CBZ, or CBR files.

Analyze & Script

AI vision models analyze the pages and generate "Did You Know" narration scripts.

Produce

Text-to-speech narration is generated and composed with the comic panels into short-form video.

Publish

Videos are posted to the operator's TikTok, YouTube, and Instagram accounts via their official APIs.

Third-Party Services Used

This application integrates with the following third-party services exclusively for content creation and publishing on the operator's own accounts:

  • TikTok Content Posting API — Publishing videos to the operator's TikTok account (@moneounchpan)
  • YouTube Data API v3 — Uploading and scheduling videos on the operator's YouTube channel
  • Instagram Graph API — Cross-posting videos as Reels on the operator's Instagram account
  • Google Cloud Text-to-Speech — Generating voice narration for videos
  • Google Gemini API — AI vision analysis and script generation
  • Cloudinary — Temporary video hosting for Instagram upload processing
