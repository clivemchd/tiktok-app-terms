Privacy Policy

Last Updated: April 1, 2026

1. Introduction

This Privacy Policy describes how the Shorts Pipeline application (“the App”) handles data. The App is a personal-use automation tool built and operated exclusively by Moneounchpan (@moneounchpan on TikTok).

Important: This application is not a public service. It has no registered users, does not accept sign-ups, and does not collect data from any third parties. The sole operator is also the sole user.

2. Information We Collect

Because this is a single-user, personal-use application, the only data processed is the operator's own authentication credentials:

  • TikTok OAuth Tokens — Used to authenticate and post video content to the operator's TikTok account via the TikTok Content Posting API.
  • YouTube OAuth Tokens — Used to upload and schedule videos on the operator's YouTube channel via the YouTube Data API v3.
  • Instagram/Meta Access Tokens — Used to cross-post videos as Reels on the operator's Instagram account via the Instagram Graph API.
  • Google Cloud Service Credentials — Used for AI vision analysis (Gemini API) and text-to-speech narration (Google Cloud TTS).
  • Cloudinary API Credentials — Used for temporary video hosting during Instagram upload processing.

We do not collect, store, or process any personal data from any other individuals. No names, email addresses, browsing history, device identifiers, IP addresses, location data, or any other personally identifiable information is collected from anyone.

3. Third-Party Tracking and Analytics

3.1 Tracking Technologies

This application and this website do not use any tracking technologies whatsoever. Specifically:

  • No cookies (other than those set by the platforms themselves when you visit this website)
  • No analytics services (no Google Analytics, no TikTok Pixel, no Meta Pixel, or any equivalent)
  • No advertising SDKs or tracking pixels
  • No fingerprinting or device identification
  • No data collection for measurement, advertising, or any commercial purpose

3.2 Third-Party Services Used

The App integrates with the following third-party services solely for content creation and publishing on the operator's own accounts. No user data is shared with these services beyond the operator's own authentication tokens and video content:

  • TikTok (ByteDance) — Video publishing via Content Posting API. Data sent: OAuth tokens, video files, post metadata (title, description). Subject to TikTok's Privacy Policy.
  • YouTube (Google) — Video uploading and scheduling via YouTube Data API v3. Data sent: OAuth tokens, video files, metadata. Subject to Google's Privacy Policy.
  • Instagram (Meta) — Video cross-posting as Reels via Instagram Graph API. Data sent: access tokens, video URLs, metadata. Subject to Instagram's Privacy Policy.
  • Google Cloud — AI analysis (Gemini API) and text-to-speech (Cloud TTS). Data sent: comic book page images for analysis, text for narration. Subject to Google Cloud Privacy Notice.
  • Cloudinary — Temporary video hosting for Instagram upload processing. Data sent: video files (temporarily). Subject to Cloudinary's Privacy Policy.

4. How We Use Information

All data processed by the App is used exclusively for the following purposes:

  • Authenticating the operator's accounts on TikTok, YouTube, and Instagram
  • Publishing video content to the operator's own social media accounts
  • Generating AI-powered narration scripts and voice audio for video content
  • Temporarily hosting video files for cross-platform publishing

No data is used for advertising, measurement, profiling, or any purpose beyond the operator's personal content publishing.

5. Data Sharing

We do not sell, trade, rent, license, or otherwise share any data with third parties for their independent use. The only data transmissions that occur are:

  • Sending video content and metadata to the platform APIs (TikTok, YouTube, Instagram) for publishing on the operator's own accounts
  • Sending text and images to Google Cloud APIs for AI processing
  • Sending video files to Cloudinary for temporary hosting

6. Data Storage and Security

All authentication tokens and credentials are stored locally on the operator's own devices and servers. Specifically:

  • OAuth tokens are stored in local JSON files on the operator's machine
  • API keys are stored in environment variables on the operator's server
  • No data is stored in any cloud database or third-party storage service
  • Video files uploaded to Cloudinary are temporary and automatically removed after processing

Reasonable security measures are implemented to protect stored credentials, including restricted file permissions and secure server access.

7. Opt-Out and Data Control

As the sole user of this application, the operator has full control over all data at all times. Additionally:

  • TikTok: The operator can revoke the App's access at any time via TikTok Security Settings. To manage ad personalization, visit TikTok Personalization and Data Settings.
  • YouTube: Access can be revoked via Google Account Permissions.
  • Instagram: Access can be revoked via Instagram account settings under Apps and Websites.
